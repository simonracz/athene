Athene
======

My sandbox for algorithms, data structures and a bunch of other stuff.

## Building
The project uses cmake. So you can browse and build the code in your favorite IDE. (http://www.cmake.org)

### Example
<pre>
athene$ mkdir build
athene$ cd build
 build$ cmake ..
 build$ make
 build$ make test
</pre>

OR

<pre>
athene$ mkdir build
athene$ cd build
 build$ cmake .. -G Xcode
 build$ open ./ATHENE.xcodeproj
</pre>
